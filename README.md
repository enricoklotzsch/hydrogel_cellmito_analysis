# Image Analysis of Cellular Morphology and Mitochondria Networks

This project analyzes cellular morphology and mitochondria networks in TIFF image stacks.

## Structure
- `data/`: Input TIFF stacks
- `scripts/`: Image processing and analysis scripts
- `notebooks/`: Example and exploratory notebooks
- `results/`: Output data and analysis results
- `visualization/`: Visualization scripts and figures

## Requirements
See requirements.txt for dependencies.

## Usage
- Place TIFF stacks in `data/`
- Run scripts in `scripts/` for processing and analysis
- Use notebooks in `notebooks/` for exploration and visualization

## Clone Notes
This repository includes `napari-mito-hcs` as a Git submodule.

- Clone with `git clone --recurse-submodules https://github.com/enricoklotzsch/hydrogel_cellmito_analysis.git`
- If you already cloned the repository, run `git submodule update --init --recursive`
